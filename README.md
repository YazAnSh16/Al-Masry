# 🎓 Masry Institute System

> A structured institute management system built using **ASP Classic**, designed to handle student and course operations efficiently while demonstrating strong understanding of backend fundamentals and legacy systems.

---

## 🚀 Project Highlights

- Built with **ASP Classic (VBScript)** — showcasing ability to work with legacy technologies  
- Implements full **CRUD operations** for core entities  
- Structured code with reusable components (includes)  
- Dynamic data rendering from database  
- Clean separation between logic and UI  

---

## 🧩 Core Features

### 👨‍🎓 Student Management
- Add, update, and delete student records  
- Display detailed student information  

### 📚 Course Management
- Manage courses  
- Assign students to courses  

### 🔗 Database Integration
- Connected to relational database  
- Execute queries dynamically  
- Handle data efficiently  

### ✅ Validation & Logic
- Input validation using JavaScript & backend checks  
- Error handling and controlled data flow  

---

## 🛠️ Tech Stack

- **ASP Classic (VBScript)**
- **HTML5 / CSS3**
- **JavaScript**
- **SQL (SQL Server / MS Access)**
- **IIS (Internet Information Services)**

---


## ⚙️ Getting Started

### 1. Requirements
- Windows OS  
- IIS enabled  
- ASP Classic feature activated  

### 2. Setup

```bash
# Move project to IIS directory
C:\inetpub\wwwroot\
```

- Configure database connection string  
- Enable ASP in IIS  

### 3. Run
